# ABC Telecom Management System

## Overview

ABC Telecom is a web-based telecom management system built with Flask and SQLite. It allows customers to register, manage their wallets, purchase airtime and data, pay bills, register their NIN, and view transaction history. It also includes a secure administrator dashboard for customer management.

## Features

### Customer Features

- User Registration
- Secure Login & Logout
- Customer Dashboard
- Wallet Management
- Buy Airtime
- Buy Data
- Bill Payment
- Transaction History
- Profile Management
- NIN Registration

### Admin Features

- Admin Login
- Dashboard
- View Customers
- Search Customers
- Customer Details
- Wallet Information
- NIN Information

## Installation

1. Clone the repository

2. Install dependencies

pip install -r requirements.txt

3. Run the application

python app.py

4. Open

http://127.0.0.1:5000
